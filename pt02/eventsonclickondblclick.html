About Clicks

onfocus: will do something when the input get the focus (on input)

onclick: Just one click (on button)

ondblclick: Double Click (on button)

onmousedown: you can click using right or left button from mouse (mouse down)

onmouseup: you can click using right or left button from mouse (mouse up)

onmousemove: you just need to move the mouse on the label (in)

onmouseout: you just need to move the mouse on the label (out)

onblur: after go out from input

onchange: after submit

oninput: after write each character

onkeypress/onkeyup/onkeydown: using key's from keyboard

onselect: after select some text

onmouseover: like on mouse move
